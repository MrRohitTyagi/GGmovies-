## Run Locally

Download the project



Create a new folder  and run the command

```bash
  npx create-react-app appname
```
Replace all files with project ones

Install Libraries 

```bash
  npm install axios
  npm install react-router-dom
```

Start the server

```bash
  npm start
```
Project will start on LocalHost:300
